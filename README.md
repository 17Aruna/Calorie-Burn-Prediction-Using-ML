# 🔥 Calorie Burn Prediction using Machine Learning

## 📌 Description
This project predicts the number of calories burned during physical activities using machine learning techniques. It analyzes user data such as age, gender, weight, height, duration, heart rate, and body temperature to estimate calorie expenditure.

## 🛠️ Technologies Used
- Python
- Pandas, NumPy
- Matplotlib / Seaborn
- Scikit-learn (Machine Learning)
- Jupyter Notebook (Anaconda)

## ⚙️ Features
- Data preprocessing and cleaning  
- Exploratory Data Analysis (EDA)  
- Feature selection and transformation  
- Model building (Regression models)  
- Prediction of calories burned  

## 📊 Model Details
- Applied regression algorithms like Linear Regression / Random Forest  
- Evaluated model using metrics such as R² Score, MAE, and RMSE
